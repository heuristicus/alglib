Do a bubble sort on the given array. The array is modified in place. Uses operator<() as the default comparison, so the ArrayType should implement that.

Parameters
arrayThe array to sort
sizeThe size of the array to sort. It is assumed that this is the actual size of the array to be sorted. A size too large may cause overflows, and a size too small will cause only the first size elements to be sorted. Negative arguments to size will throw an exception.

Do a bubble sort on the given array. The array is modified in place. The ordering is defined based on the function that is provided.

Parameters
arrayThe array to sort
sizeThe size of the array to sort. It is assumed that this is the actual size of the array to be sorted. A size too large may cause overflows, and a size too small will cause only the first size elements to be sorted. Negative arguments to size will throw an exception.
compPointer to a function or functor which returns true if the first value takes priority over the second.
